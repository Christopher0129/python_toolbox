# LLM 网关专题

这个专题关注大模型调用不应散落在各个服务里，而应有统一入口做认证、限流、路由和审计。

## 适用场景

- 多团队共享模型接入
- 多供应商、多模型并行使用
- 需要统一治理成本、配额和安全边界

## 常见主题

- 统一认证
- 模型路由
- 配额与成本控制
- Prompt / response 审计
- 失败回退

## 设计要点

- 网关的目标不是只做代理，而是沉淀治理策略
- 模型参数、超时、重试和审计字段应统一抽象
- 对敏感场景要在网关层补脱敏、日志分级和审批
- 网关要给业务方暴露清晰 SLA 和错误语义
