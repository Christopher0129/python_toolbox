# 异步测试专题

这个专题关注如何验证协程、异步数据库、异步 HTTP 客户端和后台任务逻辑，而不是把异步代码当同步函数硬测。

## 适用场景

- FastAPI / asyncio 服务测试
- 异步数据库和 HTTP 客户端测试
- 后台协程任务与事件循环资源管理

## 常见主题

- 协程测试函数声明
- 异步 fixture 生命周期
- 事件循环隔离
- 异步依赖替换与 mock
- 超时、取消和并发竞态验证

## 设计要点

- 测试必须显式管理事件循环和异步资源关闭
- 异步测试更要关注超时和任务泄漏
- 外部依赖 mock 后，也要保留关键真实集成路径
- 同步和异步调用链混用时要先厘清边界
