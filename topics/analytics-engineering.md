# 分析工程专题

这个专题关注“分析逻辑工程化”，而不是只写零散 SQL 或一次性脚本。

## 常见主题

- 模型分层
- 数据测试
- 文档和血缘
- 调度与发布
- 实验跟踪与模型治理
- 数据契约与 schema 演进

## 常见工具

- `dbt`
- `DuckDB`
- `Polars`
- `MLflow`
- `Great Expectations`
