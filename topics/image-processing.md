# 图像处理专题

这个专题关注 Python 日常图像处理任务：缩放、裁剪、格式转换和批量处理。

## 常用库

| 库 | 作用 |
| --- | --- |
| `Pillow` | 基础图像处理 |
| `pathlib` | 路径批处理 |

## 常见任务

- 批量缩略图
- PNG/JPG 格式转换
- 图片尺寸标准化
- 扫描目录后批量重命名
- 视频帧抽取和视觉前处理
