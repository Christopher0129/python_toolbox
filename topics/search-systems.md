# 搜索系统专题

这个专题关注从结构化查询走向全文检索、聚合检索和搜索相关性调优。

## 常见能力

| 能力 | 说明 |
| --- | --- |
| 倒排索引 | 文本词项快速检索 |
| 过滤与聚合 | 精确过滤和统计分析 |
| 高亮与排序 | 搜索结果展示与相关性 |
| 混合检索 | 关键词 + 向量召回组合 |
| 查询改写 | 把自然语言问题转成更稳定的检索表达 |
| 索引构建 | 管理全文索引、增量更新和重建过程 |
