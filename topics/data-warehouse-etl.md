# 数据仓库与 ETL 专题

这个专题关注从业务数据到分析数据的整理过程：抽取、清洗、装载、建模和调度。

## 常见阶段

1. 抽取数据
2. 清洗和标准化
3. 装载到分析库
4. 构建主题表或宽表
5. 调度和监控

## 常见工具

- `pandas`
- `Airflow`
- 数据库和 SQL 工具
- `dbt`
- `Great Expectations`
