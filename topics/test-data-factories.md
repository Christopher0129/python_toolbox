# 测试数据工厂专题

这个专题关注如何稳定生成测试数据，而不是在每个测试里手写一堆重复字典和对象。

## 适用场景

- ORM 模型测试
- API 请求体、响应体构造
- 复杂对象图与关联关系测试

## 常见主题

- 工厂类设计
- 伪造数据生成
- 关联对象组合
- 唯一字段与序列值
- 可读性与灵活性平衡

## 设计要点

- 工厂的目标是减少样板代码，不是隐藏业务语义
- 默认值要真实但可控，避免随机性破坏可复现
- 关联对象层级不能无限膨胀，否则维护成本会很高
- 工厂应服务测试场景，不应渗入生产代码
