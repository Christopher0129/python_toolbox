# 模型路由专题

这个专题关注不同模型之间如何按成本、时延、能力和可用性分配请求。

## 适用场景

- 同时接多个供应商或多个模型档位
- 简单请求和复杂请求希望走不同模型
- 需要回退和配额控制

## 常见主题

- 按任务类型分流
- 按成本档位分流
- 回退链
- 区域与可用性路由
- 配额与限流

## 设计要点

- 路由策略必须可解释，否则排障会很困难
- 回退要定义清楚是降级质量、降级功能还是只换供应商
- 路由决策应记录到日志和评测报表中
- 不同模型输出差异要通过 schema 和评测兜底
