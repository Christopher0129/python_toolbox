# 密码安全专题

这个专题关注密码不是“加密后存一下”，而是要用合适的哈希算法、校验流程和重置机制。

## 适用场景

- 用户账号登录体系
- 管理后台、内部平台和多租户系统
- 密码迁移、重置和凭证轮换治理

## 常见主题

- 密码哈希
- 盐和参数成本
- 登录校验
- 密码重置与失效
- 凭证泄露后的轮换策略

## 设计要点

- 密码要存哈希，不要可逆加密
- 哈希参数要结合当前硬件和登录峰值调整
- 重置令牌要有一次性、短时效和审计记录
- 登录失败计数、二次验证和风控要联动考虑
