# 服务中间件专题

这个专题关注请求在到达业务逻辑前后，如何统一处理日志、鉴权、异常和追踪信息。

## 常见职责

- 请求日志
- Trace ID 注入
- 认证校验
- 异常统一包装
- 响应耗时统计
- 统一限流与特性开关上下文
