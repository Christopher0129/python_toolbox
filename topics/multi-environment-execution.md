# 多环境执行专题

这个专题关注同一套测试、构建或脚本如何在多个 Python 版本、依赖组合和运行环境中稳定执行。

## 适用场景

- 兼容多个 Python 版本的库
- 多依赖版本验证
- CI 与本地开发环境统一

## 常见主题

- 环境矩阵定义
- 命令复用
- 构建和测试分层
- 缓存与环境重建
- 多平台差异处理

## 设计要点

- 环境矩阵要服务风险控制，而不是机械堆数量
- 测试、lint、类型检查和构建任务应可拆分复用
- 本地与 CI 的命令入口尽量保持一致
- 要区分一次性 smoke 验证和完整 release 验证
