# 列式数据处理专题

这个专题关注比传统行式脚本更适合分析任务的列式处理思路。

## 常见工具

| 工具 | 适用方向 |
| --- | --- |
| `Polars` | 高性能本地 DataFrame |
| `DuckDB` | 本地分析型 SQL |
| `Pandas` | 通用表格处理 |

## 适用场景

- 大文件分析
- 本地 OLAP
- SQL 与 DataFrame 混合分析
