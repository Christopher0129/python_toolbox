# RAG 流水线专题

这个专题关注从文档到检索增强生成的完整链路。

## 常见阶段

1. 文档加载
2. 切分与清洗
3. 向量化
4. 向量存储
5. 检索
6. 重排
7. 生成

## 常见工具

- `LangChain`
- `LlamaIndex`
- `FAISS`
- `Qdrant`
- `Milvus`
