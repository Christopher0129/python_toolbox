# 对象存储专题

这个专题关注不是把文件放在本地磁盘，而是把数据、媒体、备份和中间产物放到 S3 兼容对象存储里。

## 常见场景

- 用户上传文件
- 日志与归档文件落盘
- 模型产物和训练结果保存
- 数据湖原始分区存储

## 常见关注点

| 主题 | 说明 |
| --- | --- |
| 路径规划 | bucket / key 结构设计 |
| 权限控制 | 读写凭证和最小权限 |
| 大文件传输 | 分片上传和流式下载 |
| 生命周期 | 归档、过期、冷存储 |
