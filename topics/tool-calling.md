# 工具调用专题

这个专题关注模型如何调用外部函数、检索、数据库和业务 API，以及如何让调用过程可控。

## 适用场景

- Agent 需要读写外部系统
- 模型要访问结构化工具而不只是纯文本生成
- 希望把大模型限制在受控操作范围内

## 常见主题

- 工具 schema
- 参数校验
- 调用授权
- 失败重试
- 调用日志与审计

## 设计要点

- 工具定义应足够清晰，减少模型在参数层面猜测
- 每个工具都应定义副作用、权限边界和失败语义
- 高风险工具要做显式审批、幂等和审计
- 工具调用结果应结构化返回，便于后续决策
