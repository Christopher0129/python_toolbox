# 并行测试专题

这个专题关注如何提升测试反馈速度，同时避免并行引入的新问题。

## 常见主题

- 多 worker 测试执行
- fixture 的并行安全
- 测试隔离和共享状态问题
- 日志与临时目录分离

## 常见工具

- `pytest-xdist`
- `testcontainers`
