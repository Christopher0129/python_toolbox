# 代理与 Egress 治理专题

这个专题关注服务访问外部网络时，如何通过代理、出口控制和审计降低风险。

## 适用场景

- 服务需要访问公网 API
- 组织要求统一出口或白名单
- 爬虫、第三方接口和大模型调用需要额外治理

## 常见主题

- HTTP / HTTPS 代理
- SOCKS 代理
- 出口白名单
- 重试与代理失败切换
- 审计与流量归因

## 设计要点

- 代理配置应集中管理，不要散落在业务代码里
- 出口治理要结合超时、连接池和 DNS 行为一起看
- 使用代理后更要关注请求头、凭证和审计留痕
- 对外访问失败应区分代理问题、目标问题和本地网络问题
